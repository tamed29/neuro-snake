# Contributing to Snake Pro

First off, thanks for taking the time to contribute! It's people like you that make Snake Pro better.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tab.
- Describe the bug and include steps to reproduce.
- Mention your OS and browser.

### Suggesting Enhancements
- Open an Issue with the tag `enhancement`.
- Explain why this feature would be useful.

### Pull Requests
1. **Fork** the repository.
2. **Clone** your fork: `git clone https://github.com/[Your GitHub Username]/snake-pro.git`
3. **Create a branch** for your fix or feature: 
   - `feat/your-feature-name`
   - `fix/your-bug-name`
   - `docs/your-doc-change`
4. **Run locally**:
   - Install dependencies: `npm install`
   - Start dev server: `npm run dev`
5. **Commit your changes**: Use clear and descriptive commit messages.
6. **Push to your fork** and **Submit a Pull Request**.

## PR Checklist
- [ ] Code follows project style
- [ ] No TypeScript errors (`npm run build`)
- [ ] Feature/bug is described clearly in the PR
- [ ] PR targets the `main` branch

We welcome all contributors regardless of experience level!
