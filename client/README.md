# 🐍 Snake Pro

A high-fidelity, competitive Snake game experience built with modern web technologies. Experience smooth gameplay, real-time leaderboards, and a premium aesthetic.

## 🚀 Features
- **Modern Physics & Classic Modes**: Choose between standard grid movement or smooth "Anti-Gravity" physics.
- **Global Leaderboards**: Authenticate to compete with players worldwide and track your highest scores.
- **Dynamic Themes**: Multiple visual modes including Cyber, Professional, and more.
- **Responsive Interface**: Fully optimized for mobile and desktop with dedicated touch controls.
- **Guest Sessions**: Play immediately without an account with smart local tracking.

## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **State Management**: Zustand
- **Database / Auth**: Firebase (Firestore & Auth)
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Icons**: React Icons & Lucide React
